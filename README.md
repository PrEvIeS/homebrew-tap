# PrEvIeS Homebrew Tap

Custom Homebrew formulas for my tools.

## Usage

```bash
# Add tap
brew tap PrEvIeS/tap

# Install ytmv
brew install ytmv
```

Or install directly:

```bash
brew install PrEvIeS/tap/ytmv
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| [ytmv](https://github.com/PrEvIeS/ytmv) | Interactive CLI wizard for downloading videos and audio from 1000+ sites |
